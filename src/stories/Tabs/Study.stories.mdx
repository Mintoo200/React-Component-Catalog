# The <code>Tabs</code> Component

## Version 1

<TryBadge href="/?path=/docs/tabs-v1--default" />

<div className="subheading">API</div>

```xml
<Tabs>
  <Tab label="Tab 1">
    Content 1
  </Tab>
  <Tab label="Tab 2">
    Content 2
  </Tab>
</Tabs>
```

<div className="subheading">Advantages</div>
<ul>
  <li>Only 2 components (<code>Tabs</code> and <code>Tab</code>)</li>
  <li>All information about a tab is at the same place in the code</li>
  <li>Very simple implementation</li>
</ul>

<div className="subheading">Drawbacks</div>
<ul>
  <li><code>label</code> is prop (cf. <a href="/?path=/docs/tabs-v1--with-icons">icons as labels</a>)</li>
  <li>Not very customizable</li>
</ul>
