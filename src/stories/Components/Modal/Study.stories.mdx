```xml
<Modal {...args} isOpen={isOpen} onModalClosed={() => setIsOpen(false)}>
  <ModalButtonClose>x</ModalButtonClose>

```

## Version 1 - Title as prop

<TryBadge href="/?path=/docs/components-modal-v1-title-as-prop--default" />

<div className="subheading">API</div>

```xml
<Modal isOpen={isOpen} onClose={() => setIsOpen(false)} title="Title">
  Try clicking outside of the modal or pressing 'Escape'!
</Modal>
```

<div className="subheading">Advantages</div>
<ul>
  <li>Only 1 component</li>
  <li>Just a wrapper over <a href="/?path=/story/modal-study--page#version-2---sub-components">version 2</a></li>
</ul>

<div className="subheading">Drawbacks</div>
<ul>
  <li>Similar to <a href="/?path=/story/tabs-study--page#version-1---labels-as-props">Tabs v1</a> but without the advantages</li>
  <li><code>title</code> is prop</li>
</ul>

## Version 2 - Sub-components

<TryBadge href="/?path=/docs/components-modal-v2-sub-components--default" />

<Modal isOpen={isOpen} onClose={() => setIsOpen(false)}>
